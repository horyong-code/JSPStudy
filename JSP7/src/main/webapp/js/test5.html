var 변수 (ES6) [P.517 ~]

* 전역 변수 : 모든 스코프 (scope)에서 사용 가능한 변수
(함수 밖에서 선언 후 사용, 함수 안에서 사용할 경우 var 타입 제거) * 지역 변수 : 특정 지역에서만 사용 가능한 변수

호이스팅 : '끌어올린다'라는 개념

자바스크립트는 인터프리터 (순차적 해석)를 사용해서 코드를 해석
함수를 만나면 함수 전체를 스캔한다
만약 함수에 var 선언이 있는 경우 별도로 저장

let 타입 : 변수를 선언한 {} (블럭)에서만 사용 가능하다

- 변수의 재사용 가능 - 변수의 재선언 불가능 (지역 변수 사용 가능) - 변수 호이스팅 불가능

const 타입 : 변하지 않는 값을 저장하는 타입

- 변수의 재선언 불가능 - 변수의 재사용 불가능 - 변수 호이스팅 불가능

var타입 : 전역변수를 선언해서 사용

- 변수 재선언 가능 [가능한 사용 X] - 변수 재사용 가능 - 변수 호이스팅 가능 - 최소한의 사용을 권장함 [코드 상단 위치]
